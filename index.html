Modern Record Player
A New, but Familiar Way to Listen to Music
Project Description
A familiar twist on the classic record player, designed for the tech-savvy audiophile. By simply placing an RFID card with the song/album/playlist name on top of the player, your favorite music will be played instantly.
Skills Learned
This project was a journey of discovery, where I combined a diverse set of skills to bring my idea to life. From programming microcontrollers in C++ and practicing the art of soldering, to designing and printing in 3D, and integrating RFID technologies all while managing the project from start to finish I transformed a concept into a tangible creation.
Artistic Turtle Bot
Microcontroller-Based Servo-Driven Drawing Robot
Project Description
With the ability to draw anything you can dream up, this innovative device is your personal artist on wheels. Simply send wireless commands to its microcontroller, and watch as the robot effortlessly maneuvers its wheels and guides the pen to create stunning artwork. Whether for art, design, or fun, the limit is your imagination.
Skills Learned
This project allowed me to combine my love of technology and art. Using a version of Python better suited for microcontrollers (microPython), I could send commands wirelessly or preprogram movements to fulfill my vision. By also designing a custom PCB and cutting acrylic to specific dimensions, everything fit together seamlessly and precisely together.
Robotic Controller
Raspberry Pi Based 3-Axis Robotic Controller
Project Description
The goal of this project was to produce a commercially viable 3-axis motion controller. This entailed creating a means for the Raspberry Pi (master controller) to send commands to a secondary interepreter (slave controller) that would control the motion of an array of stepper motors. This would then allow the control of any number of machinery including 3D printers, CNC machine, laser cutter, pastry decorator machine, and more.
Skills Learned
This project was a culmination of all the skills I'd learned up to that point. Combining project development, PCB design, 3D design and printing, programming, soldering, laser cutting, and countless other considerations to build something commercially viable.
Hovercraft
Self-Stabilizing Arduino Based Hovercraft
Project Description
The objective of the project was to design and build a fully autonomous hovercraft capable of navigating a track. The track includeded three obstacles of increasing height and would vary in width. The hovercraft must successfully traverse the course, stop precisely under a horizontal bar at the end.
Skills Learned
This project involved working and collaborating with a team to design and build a hovercraft with a variety of sensors and motors. Since it was designed to be self-stabilizing and autonomous, it required a lot of testing and debugging to ensure it could navigate the course.
